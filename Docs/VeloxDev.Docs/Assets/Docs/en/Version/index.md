# Version

Flip through this chapter, and you will quickly learn about historically important version updates. The document pruning rules are as follows:

| Condition | Behavior |
| --- | --- |
| Modify the usage of an existing feature block | Create a subpage in the [Version] section to describe new content, and annotate the version of the change in the corresponding original text |
| New feature | Add directly in the original text |
| Delete existing feature block | Delete the corresponding original text |
