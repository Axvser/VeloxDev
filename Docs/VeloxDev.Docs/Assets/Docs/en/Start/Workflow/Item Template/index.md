# Item Template

VeloxDev has provided item template support for some platforms that support additional actions, which can be used to efficiently build workflow views. This chapter demonstrates its basic command line usage and key points for fine-tuning template code. Details about additional attributes can be found in the next section.

> **Install - {$plateform} Optional: Avalonia / WPF / WinUI / MAUI**

```shellscript
dotnet new install VeloxDev.{$plateform}.Templates
```
