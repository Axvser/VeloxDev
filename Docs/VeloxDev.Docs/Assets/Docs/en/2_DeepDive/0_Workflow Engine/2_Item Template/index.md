# Item Template

The template system generates platform-specific Views for each component type.

```shell
dotnet new {platform}-v-{component}
```

| Template | Purpose |
|----------|---------|
| `ava-v-tree` | Tree View for Avalonia |
| `wpf-v-node` | Node View for WPF |
| `winui-v-slot` | Slot View for WinUI |
| `maui-v-link` | Link View for MAUI |
