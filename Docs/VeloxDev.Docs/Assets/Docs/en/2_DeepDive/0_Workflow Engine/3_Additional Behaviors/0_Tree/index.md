# Tree Surface Behavior

`WorkflowSurfaceBehavior` enables the Tree's canvas with pan, zoom, and grid snap.

**Attached Properties:**

| Property | Type | Description |
|----------|------|-------------|
| `IsEnabled` | `bool` | Enable/disable the behavior |
| `ScrollViewerName` | `string` | Name of the ScrollViewer element |
| `CanvasName` | `string` | Name of the Canvas element |
| `GridDecoratorName` | `string` | Name of the GridDecorator element |
| `PointerPressSourceName` | `string` | Name of the pointer-sensing element |
| `MinimapOverlayName` | `string` | Name of the minimap overlay element |
