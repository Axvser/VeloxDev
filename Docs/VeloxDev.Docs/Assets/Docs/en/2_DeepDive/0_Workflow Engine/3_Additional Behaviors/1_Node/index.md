# Node Interaction

Nodes respond to standard commands wired through the ViewModel:

- **Move**: Drag via `MoveCommand(Offset)` — handled by `Helper.Move()`
- **SetAnchor**: Right-click placement via `SetAnchorCommand(Anchor)`
- **Delete**: Remove node via `DeleteCommand()` — calls `Helper.Delete()`
- **CreateSlot**: Add a new slot via `CreateSlotCommand(SlotViewModelBase)`

These are automatically generated by the source generator from the `[VeloxCommand]` annotations on `NodeViewModelBase`.
