# Tree 表面行为

`WorkflowSurfaceBehavior` 为 Tree 画布提供平移、缩放、网格吸附能力。

**附加属性：**

| 属性 | 类型 | 描述 |
|------|------|------|
| `IsEnabled` | `bool` | 启用/禁用行为 |
| `ScrollViewerName` | `string` | ScrollViewer 元素名称 |
| `CanvasName` | `string` | Canvas 元素名称 |
| `GridDecoratorName` | `string` | GridDecorator 元素名称 |
| `MinimapOverlayName` | `string` | 小地图覆盖元素名称 |
