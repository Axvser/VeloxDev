## 📐 坐标系

工作流画布使用**标准计算机图形坐标系**（屏幕坐标系）：

- **原点 (0, 0)** = 画布左上角。
- **Anchor.Horizontal（X）** 向**右**增大（左 → 右）。
- **Anchor.Vertical（Y）** 向**下**增大（上 → 下）。
- **Anchor.Layer（Z）** = 层叠顺序（值越大越靠前显示）。

```
  (0,0) ────── X+ (Horizontal) ──────►
    │
    │
   Y+ (Vertical)
    │
    ▼
```

### 自然语言坐标转换

当用户以自然语言描述位置时，请转换到此坐标系：

| 用户表述 | 含义 |
|---|---|
| "左上角"、"左上" | 小 X，小 Y（靠近原点） |
| "右上角"、"右上" | 大 X，小 Y |
| "左下角"、"左下" | 小 X，大 Y |
| "右下角"、"右下" | 大 X，大 Y |
| "中间"、"居中" | 所有现有节点包围盒的中心点 |
| "X 节点上方" | 与 X 相同的水平位置，**更小**的 Y |
| "X 节点下方" | 与 X 相同的水平位置，**更大**的 Y |
| "X 节点左边" | **更小**的 X，与 X 相同的垂直位置 |
| "X 节点右边" | **更大**的 X，与 X 相同的垂直位置 |
| "向上移动" | **减小** Y（负 offsetY） |
| "向下移动" | **增大** Y（正 offsetY） |
| "向左移动" | **减小** X（负 offsetX） |
| "向右移动" | **增大** X（正 offsetX） |

⚠️ **常见陷阱**："上方"意味着 Y **减小**（趋向 0），而非增大。"下方"意味着 Y **增大**（远离 0）。
