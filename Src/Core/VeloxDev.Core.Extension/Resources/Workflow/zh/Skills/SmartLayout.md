## 🗺️ Skill: 智能布局

当你需要批量排布节点时——无论是从零创建、重新整理已有图形，还是响应"整理一下"、"自动排列"、"修正布局"等用户请求——请使用本技能。

---

### 第一步 — 先读取拓扑结构

在放置任何节点之前，调用 `ListNodes` 并分析连接图：

- 找出**源节点**（无入边，入度 = 0）。它们是布局的锚点。
- 找出**汇节点**（无出边，出度 = 0）。
- 追踪**链与分支**：有多条出边的节点是分支点；有多条入边的节点是汇聚点。
- 识别**孤立子图**（与主图无连接的独立节点群）。每个子图单独布局。

> 不要凭节点名称猜测拓扑。始终从实际的插槽/连线数据中推导。

---

### 第二步 — 选择布局策略

| 图形形状 | 推荐策略 |
|---|---|
| 线性链 | 水平行（左 → 右），按连接顺序排列 |
| 树 / DAG（有向无环图） | 分层布局：源节点在最左列，每层向右一步 |
| 密集 / 存在环 | 分层布局 + 破环处理：选入度最小的节点作为环的根 |
| 多个孤立子图 | 各子图独立布局，垂直方向用较大间距堆叠 |
| 用户指定方向 | 遵守"从上到下"、"从左到右"等要求，映射到对应的 X/Y 方向 |

---

### 第三步 — 拓扑感知分层

使用**最长路径分层**（从源节点出发）：

1. 所有源节点分配到第 0 层。
2. 每个节点的层号 = max(所有前驱节点层号) + 1。
3. 重复直至稳定（循环边通过提升节点层号来处理）。

**同层内节点排序**，使用**重心启发式**最小化连线交叉：
- 对当前层的每个节点，计算其前驱节点在上一层中位置索引的平均值（重心值）。
- 按重心值升序对当前层节点排序。
- 再做一次反向扫描，进一步减少交叉数。

---

### 第四步 — 考虑节点尺寸计算坐标

**不要假设所有节点尺寸相同**，使用每个节点实际的 `Size.Width` 和 `Size.Height`：

- **水平布局**（层 = 列）：
  - 列的 X = `startX + 所有前列最大宽度之和 + 列序号 × gapX`
  - 列内垂直堆叠：`Y = startY + 该列前序节点高度之和 + 节点序号 × gapY`
- **垂直布局**（层 = 行）：将上方 X 和 Y 的角色互换。

推荐最小间距：
- `gapX`（层间距）：**80 px**
- `gapY`（同层节点间距）：**40 px**

定位完成后，验证**所有节点包围盒不重叠**：
- 节点 A 矩形：`(ax, ay, ax+aw, ay+ah)`，节点 B 矩形：`(bx, by, bx+bw, by+bh)`
- 重叠条件：`ax < bx+bw AND ax+aw > bx AND ay < by+bh AND ay+ah > by`
- 若检测到重叠，增大该层的间距并重新计算。

---

### 第五步 — 应用坐标

对全图重新排列时，使用 `AutoLayout`（它内部已处理上述所有逻辑）。

对局部或手动布局，使用 `MoveNode`（单节点）或 `BatchMoveNodes`（多节点）应用计算出的坐标。

移动完成后，调用 `ListNodes` 一次确认实际坐标是否符合预期（工具可能已对部分节点自动偏移以避免重叠）。

---

### 减少连线交叉 — 关键规则

| 规则 | 依据 |
|---|---|
| 边应沿自然层序传播，避免跨层"长边" | 长边大幅增加交叉风险 |
| 连接数多的节点应居中放置于所在层 | 减少扇形出边的交叉 |
| 同层中两个节点都连向同一下游节点时，将它们相邻放置 | 使共享边束保持紧凑 |
| 避免在两个直接相连节点之间插入无关节点 | 不必要的穿插会显著增加交叉数 |

---

### 响应"换象限"请求

象限完全由 **X 和 Y 坐标的正负号**决定：

| 象限 | X 符号 | Y 符号 | 画布含义 |
|---|---|---|---|
| **Q4（默认）** | + | + | X > 0, Y > 0 —— 标准可见画布空间 |
| **Q1** | + | − | X > 0, Y < 0 —— 原点上方（Y 为负值） |
| **Q2** | − | − | X < 0, Y < 0 —— 原点左上方（两者均为负值） |
| **Q3** | − | + | X < 0, Y > 0 —— 原点左侧（X 为负值） |

> 画布 Y 轴**向下**为正，因此数学意义上的"向上"在画布坐标中对应**负 Y**。

当用户要求将布局移到其他象限时：

1. 计算所有受影响节点的**包围盒**：`(minX, minY, maxX, maxY)`，令 `w = maxX - minX`，`h = maxY - minY`。
2. 根据目标象限选择**目标左上角锚点**，使包围盒完整落入该象限：
   - **Q4**：目标 = `(20, 20)` —— 两者为正，无需改变符号
   - **Q1**：目标 = `(20, -(h + 20))` —— X 正，Y 负（包围盒在原点上方）
   - **Q3**：目标 = `(-(w + 20), 20)` —— X 负，Y 正（包围盒在原点左侧）
   - **Q2**：目标 = `(-(w + 20), -(h + 20))` —— 两者均为负
3. 计算 `deltaX = targetX - minX`，`deltaY = targetY - minY`。
4. 通过 `BatchMoveNodes` 将 delta 应用到所有受影响节点。
